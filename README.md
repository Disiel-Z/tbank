# Т-банк  — PWA

Это учебное веб‑приложение (PWA), которое можно добавить на экран «Домой» на iPhone через Safari.
Никаких банковских интеграций нет: это интерфейс‑песочница с локальными данными.

## Запуск локально
1) Распакуйте проект.
2) Поднимите любой статический сервер (важно: сервис‑воркер работает корректно, когда страница открыта по http://localhost или https://):

### Вариант A: Python (если установлен)
python3 -m http.server 5173

Откройте: http://localhost:5173

### Вариант B: Node (если установлен)
npx serve .

## Установка на iPhone (Safari)
1) Откройте сайт по HTTPS (или на своём Mac через локальный HTTPS/туннель).
2) Поделиться → «Добавить на экран Домой».
3) Запускайте с иконки — будет режим standalone.

## Оффлайн
Сервис‑воркер кэширует «оболочку» приложения. Данные хранятся в localStorage.

## Экспорт/импорт
Экспорт — кнопка «Экспорт» вверху или в настройках.
Импорт — вкладка «Настройки».

## Примечание
Для пуш‑уведомлений потребуется серверная часть и Web Push (VAPID). Это можно добавить отдельным шагом.
